# 💊 Medication Information Translator

## Description

Medication Information Translator is a Python application that helps users search for medication information and understand it in simple English.

The application retrieves drug information from the openFDA API, checks medication details, stores search history, and provides an easy explanation.

## Features

✅ Search medication information  
✅ Usage information  
✅ Warnings  
✅ Side effects  
✅ Dosage instructions  
✅ FDA recall checking  
✅ AI simple explanation  
✅ Search history storage  

## Technologies Used

- Python
- Streamlit
- openFDA API
- Gemini AI
- JSON File Handling

## Python Concepts Demonstrated

- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
- Regular Expressions
- API Integration
